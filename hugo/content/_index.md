Welcome to metadatahub.eu, the linked data hub for cultural heritage metadata standards, vocabularies, and dataset catalogs operated by Port 30 KB.

This hub serves both **humans** (browsing documentation, specifications, and guides) and **machines** (requesting RDF data via content negotiation, SPARQL, and OAI-PMH) from the same URIs, following W3C linked data best practices.
